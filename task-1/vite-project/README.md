# Reusable Form System with Scroll Navigation

A single-page React + TypeScript application featuring a multi-section form with a scroll-synced sidebar navigation, built with Formik, Vite, and a dark editorial design system.

---


## Prerequisites

Make sure you have the following installed:

| Tool | Minimum Version | Check |
|------|----------------|-------|
| Node.js | v18+ | `node -v` |
| npm | v9+ | `npm -v` |

---

## Getting Started

cd vite-project
```

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

The app will be available at **http://localhost:5173** by default. 

---
