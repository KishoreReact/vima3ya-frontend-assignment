# VOID³ — 3D Model Viewer

A browser-based GLB model viewer built with Three.js, featuring Draco decompression, lazy loading, and proper memory management.

---

## Requirements

- Node.js 18+
- npm 9+

---

## Setup & Run

```bash

cd 3d-model-viewer

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
```

Then open http://localhost:3000 in your browser.

---

## Build for Production

```bash
npm run build     # outputs to /dist
npm run preview   # serves the production build locally
```
